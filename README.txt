Médiateur.com - Pack complet (prototype)
Installation:
1. Copier les fichiers dans le dossier public/www de votre serveur.
2. Importer schema.sql dans MySQL.
3. Mettre à jour config.php avec vos accès DB.
4. Créer un admin initial en insérant un utilisateur dans la table users (utiliser password_hash en PHP).
